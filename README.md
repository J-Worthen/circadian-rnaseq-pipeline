# Circadian RNA-seq Pipeline

This repository contains scripts used for RNA-seq preprocessing, alignment, transcript quantification, and selected downstream motif-site processing for a circadian adipocyte transcriptomics workflow.

## Workflow overview

1. rRNA removal (SortMeRNA)
2. Alignment to mouse genome (STAR)
3. Transcript quantification (RSEM)
4. Batch correction (LIMBR)
5. Circadian analysis (ECHO)
6. Motif analysis of rhythmic chromatin regions

## Requirements

- SortMeRNA
- STAR
- RSEM
- R (motifmatchr, TFBSTools, BSgenome)
- LIMBR
- ECHO

## Batch correction and circadian analysis

Transcript-level TPM matrices generated by RSEM were batch-corrected using LIMBR (Crowell et al., 2019) following the standard workflow. Default parameters were used unless otherwise specified.

Circadian rhythmicity was assessed using ECHO (De los Santos et al., 2020) with the following parameters:
- Period range: 22–30 hours  
- Multiple testing correction: Benjamini–Hochberg  
- Significance threshold: adjusted p-value ≤ 0.005  
- Allowed oscillation types: harmonic, damped, and forced  

LIMBR and ECHO were run using their publicly available implementations:
- LIMBR: https://github.com/aleccrowell/LIMBR  
- ECHO: https://github.com/delosangeles/ECHO

## Notes

- Paths are written relative to the repository root using `$(pwd)` or `getwd()`.
- Reference FASTA/GTF files are expected in `references/`.
- STAR genome indexing and RSEM reference preparation require substantial memory for mammalian genomes and should be run on an HPC or high-memory workstation.
- Large generated outputs, STAR indexes, RSEM references, FASTQ files, and BAM files should not be committed to GitHub.

## Motif enrichment code availability

Motif enrichment statistics were generated using a Dartmouth College Data Analytics Core developed workflow that is not redistributed here. The repository includes the motif set and downstream scripts used to scan motif locations, filter enriched motif sites by genomic context, restrict sites to circadian genes, and summarize early-wave target genes from the enrichment output tables.
